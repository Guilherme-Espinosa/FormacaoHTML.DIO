# FormacaoHTML.DIO
Um diretório criado para registrar o avanço na Formação HTML Web Developer - DIO. Uma oportunidade de massificar os conhecimentos que são base para criar as páginas WEB. Aprender na prática a como estruturar a marcação da página e como funciona cada tag, conheçendo os elementos semânticos e outras boas práticas na hora de construir as páginas web.
